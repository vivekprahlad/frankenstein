Frankenstein is a Functional Testing tool for Swing applications. Frankenstein's focus is on readable, simple, fast functional tests that can be shared and run by everyone on a team.

Apart from automating your functional tests, you could also use Frankenstein for recording bugs so that they may be easily reproduced later.

Features

  • Record and play support
  • Supports most swing components
  • Support for testing multithreaded applications
  • Test scripts are independent of widget location
  • Easy integration
  • Easy customization / extension
  • HTML reports. (Screenshots are taken when a test step fails. Here is a sample report)
  • Ruby driver
  • OGNL based assertions
  • Designed to minimize the need for explicit delays in tests
  • Supports JDK 1.4+
